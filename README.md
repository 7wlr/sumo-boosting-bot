# Sumo boosting bot

Logs in queue's sumo and when the game started it will requeue

## Setup

1. Clone the repo:
   ```bash
   git clone https://github.com/7wlr/sumo-boosting-bot.git
   cd mineflayer-bots
   ```

2. Install dependencies:
   ```bash
   npm i mineflayer
   ```

3. Update `accounts` in `boost.js` with your usernames.

4. Run:
   ```bash
   node boost.js
   ```
